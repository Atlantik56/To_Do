# Планировщик рабочих задач

Современное одностраничное приложение для управления рабочими задачами в формате канбан-доски. Проект сделан на React, Vite и чистом CSS с CSS Modules.

## Демо

После успешного выполнения GitHub Actions приложение будет доступно на GitHub Pages:

https://atlantik56.github.io/To_Do/

## Возможности

- Создание, просмотр, редактирование и удаление задач.
- Поля задачи: название, описание, приоритет и статус.
- Три статуса канбан-доски: `К выполнению`, `В процессе`, `Готово`.
- Три уровня приоритета: `Высокий`, `Средний`, `Низкий`.
- Поиск задач по названию.
- Фильтрация по статусу и приоритету.
- Автоматическое сохранение задач в `localStorage`.
- Адаптивная темная тема.
- CSS-анимации для появления и удаления карточек.

## Технологии

- React
- JavaScript
- Vite
- CSS Modules
- GitHub Actions
- GitHub Pages

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Production-сборка

```bash
pnpm build
```

Для GitHub Pages сборка выполняется с base path репозитория:

```bash
pnpm build -- --base=/To_Do/
```

## Деплой

Деплой выполняется автоматически через GitHub Actions при каждом push в ветку `main`.

Workflow находится здесь:

```text
.github/workflows/vite-build.yml
```

Он устанавливает зависимости, собирает Vite-приложение и публикует папку `dist` на GitHub Pages.
